Le Web : introduction

Définition : Service proposé sur le réseau Internet permettant de naviguer sur des pages en cliquant sur des liens. On parle aussi de World Wide Web (toile d’araignée mondiale).

Repères historique

Le Web a été développé au CERN (Conseil Européen pour la Recherche Nucléaire) par le Britannique Sir Timothy John Berners-Lee et le Belge Robert Cailliau au début des années 90. À cette époque les principaux centres de recherche mondiaux étaient déjà connectés les uns aux autres, mais pour faciliter les échanges d’information Tim Berners-Lee met au point le système hypertexte. Ce système permet, à partir d’un document, de consulter d’autres documents en cliquant sur des mots clés. Ces mots "cliquables" sont appelés hyperliens. Le tout premier document hypertexte (page web), est toujours accessible à l’adresse :

http://info.cern.ch/hypertext/WWW/TheProject.html

Navigateurs web

Pour visualiser les pages web et utiliser les hypertextes, Timothy John Berners-Lee développe le premier navigateur web qu’il appelle simplement WorldWideWeb.

Définition : Un navigateur est un logiciel permettant de visualiser une page web et d’utiliser les hypertextes.

Fonctionnement

D'un point de vue technique, le web est fondé sur 3 éléments :

  • URL : Uniform Resource Locator est un texte permettant de localiser une ressource sur un serveur et plus généralement sur internet
  • HTTP(s) : HyperText Transfer Protocol (Secure) est un protocole de communications client-server.
  • HTML : Hypertext Markup Language est un langage de programmation permettant d’écrire des documents hypertextes (pages internet)

L'URL a déja été abordée à l'occasion du thème internet ici. Le protocole HTTP et le langage HTML seront abordés lors des prochaines parties.
