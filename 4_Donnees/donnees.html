Les données

Qu’est ce qu’une donnée ?

Définition : Une donnée est une valeur (texte, nombre, date, ...) attribuée à une entité pour la décrire.

Les données peuvent prendre différentes formes. Elles sont ensuite traduites en binaire (uniquement en 0 et 1) pour être stockées sur des supports numériques dans un fichier. Le format du fichier, principalement visible grâce à son extension, permet de reconstruire les données à patir du binaire.

Pourquoi tant de format ?

Les formats répondent à différentes problématiques :

  • Mémoire : lorsque les données sont comprésée pour utiliser moins de place sur le disque, cette compression peut se faire avec pertes (mp3, jpeg par exemples) pour être plus efficace ou sans pertes (flac, png par exemples). L'algorithme de compression choisit a aussi un impact sur le temps de calcul necessaire à cette compression.
  • Modifications. Certaines données doivent être facilement modifiables (documents textes doc(x) ou odt) ou à l’inverse sécurisées (pdf avec options de sécurités).
  • Licences. Certains formats sont basés sur des avancés développée par des entreprises qui veulent garder le contrôle sur ces formats. Les formats sont alors « propriétaires » ou « fermés » (mp3, doc(x) par exemple). Par oppositions, les autres formats sont dits « ouverts » ou « libre » (ogg ou odt par exemple).

Le binaire

Les supports numériques sont des supports contenant des millions de petits emplacements. Chaque petit emplacement peut adopter deux positions différentes. Par exemple, un cd contient des petites cavités qui réfléchissent la lumière. Au moment de graver le cd, certaines cavités sont brûlées et changent ainsi de position : elles ne réfléchissent plus la lumière. Au moment de la lecture du cd, la tête de lecture éclaire les cavités une par une et mesure la lumière réfléchie, déterminant ainsi si la cavité a été brûlée ou non.

Une des positions est associée à la valeur 1, l’autre à la valeur 0. On peut alors écrire des nombres en utilisant uniquement ces deux chiffres, c’est le « binaire ». Un seul chiffre (0 ou 1) est appelé un « bit ». Le regroupement de 8 bits est appelé un « octet». Un disque dur par exemple, contient des milliards de bits. Ces bits peuvent représenter des données de différentes natures, et hors contexte, il est très difficile de déterminer quelle est la nature de ces données.

Pour stocker du texte brut (un programme Python par exemple), un des encodages en binaire utilisé est l’ASCII. Cet encodage permet de coder tous les caractères utilisés en langue anglaise. Chaque caractères est traduit par une série de 7 bits. Collecte de données

La collecte de données se fait en partie par de la saisie de données par un être humain. Exemples :

  • formulaire d'inscription à l'université
  • dossiers administratifs

Mais la collecte peut aussi être automatisée. Exemples :

  • vidéo-surveillance
  • télé-péages
  • pisteurs sur internet pour collecter les sites visités

Ces données ont un prix. Même si la gestion de ces données est très encadrées, la revente de bases de données génèrent des millions de dollars chaque année. Des hackers se sont même spécialisés et revendent sur le marché illégal des bases de données obtenu par piratage.

Big Data

Aujourd'hui, le volume des données collectées double tous les 3 ans. La raison : les réseaux sociaux collectent beaucoup d'informations sur leurs milliards d'abonnés. Les grands centres de recherche disposent aussi d'instrument collectant des données en très grand nombre. Le grand collisionneur de hadrons du CERN collecte 1 Mo pour chaque collision entre deux particules, il s'en produit environ 600 par secondes.

Les big data sont définis par les 3V :

  • Volume : un volume de données de plus en plus conséquent
  • Variété : des données de natures différentes
  • Vélocité : des données qui évoluent très vite

Une nouvelle branche de l'étude des données est apparut car les méthodes conventionnelles ne peuvent pas s'adapter aux big data.
