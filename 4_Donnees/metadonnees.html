Les métadonnées

Une métadonnée est une donnée servant à définir ou décrire une autre donnée, quel qu'en soit le support (papier, électronique ou autre). On peut citer par exemple la date à laquelle elle a été produite ou enregistrée une donnée.

Pour un fichier audio

Les logiciels de gestion de bibliothéque musicales exploitent les métadonnées des fichiers des morceaux de musique pour en extraire les informations sur l'artiste, l'album, le genre musical...

  1. Télécharger le fichier Meydn-SynthwaveVibe.mp3
  2. Afficher les propriétés du fichier (clic-droit / propriétés)
  3. Quel est le genre musical de ce morceaux ?
  4. En quelle année ce morceaux a-t-il été composé ?

Pour un site web

Lorsque l'on écrit une page web, ces métadonnées peuvent être ajouter grâce aux balises meta. Elle permettent alors d'indiquer la nature de l'information ajoutée ainsi que sa valeur.

  1. Rendez-vous sur le site www.moma.org/
  2. Afficher le code source de la page (clic-droit / afficher le code source)
  3. Quelle est la valeur associée à la métadonnées décrivant le nom du site (og:site_name) ?

Sur Twitter

Les métadonnées sont présents quotidiennement. Par exemples, des métadonnées accompagne chaque tweet posté. Twitter fournis des exemples de métadonnées dans sa documentation pour les développeurs.

  1. Rendez-vous sur le site developer.twitter.com/en/docs/twitter-api/v1/data-dictionary/object-model/example-payloads
  2. D'après cette documentation, les volumes concernant les métadonnées est-il supérieur au volume du tweet en lui même (140 caractères maximum)

Pour une image numérique

Les métadonnées accompagnent aussi les photos prises grâce à un appareil photo numérique ou grâce à un smartphone. Le format de ces métadonnées est unifié sous le nom de données EXIF que nous avons déjà abordé ici.
