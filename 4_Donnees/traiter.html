Traiter des données structurées

Bases de données

Une base de données est la collection de données décrivant des entités de même nature. Les données sont souvent représentées en tableau. Chaque ligne correspond à une des entités, on parle d’enregistrement. Le nom d’une colonne est un descripteur. Pour visualiser et manipuler ces données, on peut adopter différentes approches :

  • utiliser un format simple (texte brut) et un langage de programmation ;
  • utiliser un tableur ;
  • utiliser un logiciel de base de données relationnelles.

Base de données en texte brut

Un moyen simple d’enregistrer les données est d’utiliser un format texte brut. Enregistrer les données en texte plutôt que directement en binaire permet de faciliter leur consultation, les échanges et le traitement par des logiciels. Les principaux formats de bases de données en texte brut sont :

Les fichiers CSV

Dans le format de fichier CSV (comma-separated values), chaque enregistrement est décrit par une ligne où les valeurs sont séparés par un symbole bien défini. Souvent ce symbole est la virgule (comma en anglais), mais parfois le point-virgule ou la tabulation. La première ligne contient généralement le nom des descripteurs.

nom;prénom;poste
Lloris;Hugo;gardien
Hernandez;Lucas;défenseur
Kimpembe;Presnel;défenseur

Le principal objectif est d’économiser l’espace nécessaire pour enregistrer les données. La conséquence est le manque de lisibilité d’un enregistrement si le nombre de descripteurs est important.

Les fichiers XML

Le format XML permet de détailler les valeurs prises par un enregistrement en les découpant selon plusieurs descripteur. Il permet alors de décrire une hiérarchie dans des entités de nature différente. La syntaxe s’apparente à celle du langage HTML.

<equipe>
    <pays>France</pays>
    <joueur>
        <nom>Lloris</nom>
        <prenom>Hugo</prenom>
        <poste>gardien</poste>
    </joueur>
    <joueur>
        <nom>Hernandez</nom>
        <prenom>Lucas</prenom>
        <poste>défenseur</poste>
    </joueur>
</equipe>

Le format JSON

Le format JSON (JavaScript Object Notation) utilise la syntaxe du langage de programmation JavaScript très utilisé sur le Web. Il permet alors une intégration très facile avec ce langage de programmation mais est aussi très facilement exploitable par les autres langages de programmation.

[
    { "nom":"Lloris", "prénom":"Hugo", "poste":"gardien" },
    { "nom":"Hernandez", "prénom":"Lucas", "poste":"défenseur" },
    { "nom":"Kimpembe", "prénom":"Presnel", "poste":"défenseur" }
]

Exploitation avec un tableur

Comme les données structurées peuvent être décrites par des tableaux, le tableur est un logiciel très indiqué pour consulter et traiter ces données. Ce type de logiciel gère d’ailleurs très bien le format CSV et propose de choisir les paramètres afin de reconstituer le tableau représentant ces données.

  2. Ouvrir ce fichier avec un éditeur de texte.
    1. Combien d’enregistrements contient cette base de données ?
    2. A quoi correspondent les trois premières valeurs de chaque enregistrement ?
    3. Comment retrouver la ligne correspondant à notre lycée ?
  3. Ouvrir ce fichier avec un tableur.
    1. Dans quelle colonne peut-on trouver le pourcentage de réussite toutes filières confondues ?

    4. Sélectionner les données, puis avec l'outil trier, ranger les établissements par taux de réussite (toutes filières confondues) puis par nombre de présents (toutes filières confondues)
